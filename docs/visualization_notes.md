# Visualization Notes

## RViz Visualization

RViz was successfully launched in a ROS Noetic Docker container with WSLg display support.

The `room_01` sequence was used for RViz visualization because it provides valid ground-truth quaternion values and was used as the main quantitative evaluation sequence.

Displayed topics:

* `/Laser_map`
* `/path`

The RViz screenshots show the local point-cloud map and the estimated trajectory generated by LOG-LIO.

## Figure Files

* `results/figures/room_01_rviz_map_1.png`
* `results/figures/room_01_rviz_map_2.png`

These figures are used as visual evidence that LOG-LIO was not only evaluated from saved trajectory files, but also visualized in a ROS/RViz environment.
